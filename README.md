# Ashwinkumar's Portfolio

Welcome to my personal portfolio! This is a modern, interactive web application built to showcase my projects, skills, and professional journey.

## 🚀 Features

- **Modern UI/UX**: A sleek dark theme with a vibrant color palette.
- **Interactive Holographic Background**: A custom-built, responsive particle canvas background that reacts dynamically to mouse interactions.
- **Terminal Typewriter Effect**: A tech-inspired auto-typing hero section (`> sudo make_it_happen`).
- **Responsive Design**: Fully mobile-friendly and optimized for different screen sizes.
- **Smooth Animations**: Integrated scroll reveal animations and a custom fluid cursor for enhanced user experience.

## 🛠️ Technologies Used

- **Framework**: React.js
- **Build Tool**: Vite
- **Styling**: Custom CSS3, Bootstrap grid layout
- **Deployment**: (Ready for Vercel / Netlify / GitHub Pages)

## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ashwin2209/MY_Portfolio.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd portfolio
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **View in browser:**
   Open `http://localhost:5173/` in your browser.
